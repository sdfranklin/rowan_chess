# Magnet Knights

Static browser version of the latest standard Magnet Knights rules.

What is included:
- 7 playable seam rows on a 5-column magnetic tile board
- standard rules only
- home row on line 5 from each side's perspective
- hidden opening selection
- human vs AI and hotseat play
- magnetic-tile visual theme in HTML/CSS/canvas

Files:
- `index.html`: page shell
- `styles.css`: layout and visual design
- `logic.js`: game rules and AI
- `app.js`: canvas rendering, input, setup flow, and game loop

For GitHub Pages:
- publish from the repository root
- `.nojekyll` is included so the static files are served directly
